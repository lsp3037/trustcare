# Project Conventions

## Git Workflow
- Always create a new dedicated branch for major code changes.
- Branch name format should follow: `feature/[task-slug]` or `fix/[bug-slug]`.
- **CRITICAL:** Nunca realize commits ou pushes automáticos para o GitHub após aplicar edições. Mantenha as modificações apenas locais para testes e validação; faça o commit/push apenas quando o usuário solicitar explicitamente (ex: comando `git push`).
